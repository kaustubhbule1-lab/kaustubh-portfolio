🌐 Web Development

7-day journey through HTML, CSS, and JavaScript — building from basics to interactive web projects.

Day 1 HTML Basics — Structure & Tags ▼

What I Learned

  • HTML Structure: DOCTYPE, html, head, body tags
  • Headings & Paragraphs: h1–h6, p tags
  • Links & Images: anchor href, img src and alt
  • Lists: Ordered (ol) and unordered (ul) with li
  • Tables: table, tr, th, td for structured data
  • Semantic HTML: Importance of meaningful tags
Day 2 HTML Forms — User Input Elements ▼

What I Learned

  • Form Elements: input, textarea, select, button
  • Input Types: text, password, email, date, file
  • Radio & Checkbox: Single and multiple selection
  • Dropdown: select with multiple option elements
  • Form Validation: required attribute and constraints
  • Labels: Associating labels with inputs
Day 3 CSS Basics — Styling & Selectors ▼

What I Learned

  • CSS Selectors: Element, Class (.class), ID (#id)
  • Colors: Names, hex codes, and rgb() values
  • Typography: font-family, size, weight, text-align
  • Box Model: margin, padding, border properties
  • Specificity: Which CSS rules take priority
  • Inline vs External CSS: Different styling approaches
Day 4 CSS Layout — Flexbox & Positioning ▼

What I Learned

  • Flexbox Container: display:flex, flex-direction, justify-content
  • Flex Items: flex-grow, flex-shrink, flex-basis
  • Space Distribution: space-between, space-around, space-evenly
  • Wrapping: flex-wrap for responsive layouts
  • Alignment: Centering elements both directions
  • Practical Layout: Built multi-column box layout
Day 5 HTML Project — Virat Kohli Documentary Page ▼

What I Learned

  • Page Structure: Building a multi-section webpage
  • Navigation Bar: Anchor-based navigation links
  • Content Layout: Intro, career highlights, image sections
  • Applying Styles: Background, fonts, margin/padding
  • Embedding Images: img tags with sizing and positioning
  • Project Practice: Combining HTML + CSS in a real page
Day 6 JavaScript Basics — Variables & Data Types ▼

What I Learned

  • Variables: var, let, const and their differences
  • Data Types: String, Number, Boolean in JS
  • String Operations: Concatenation and type coercion
  • Boolean Arithmetic: true/false addition and multiplication
  • typeof Operator: Checking data type of a variable
  • console.log: Printing output to the browser console
Day 7 JavaScript Project — To-Do List App ▼

What I Learned

  • DOM Manipulation: Selecting and modifying HTML elements
  • Event Listeners: Adding click events to buttons
  • Dynamic Content: Creating and appending new list items
  • Input Handling: Reading user input from text fields
  • Project Integration: HTML + CSS + JS in one working app
  • User Experience: Building an interactive web application
